**Текст для копирования в Replit AI:**
> Напиши Telegram-бота для техподдержки на языке Python, используя библиотеку pyTelegramBotAPI (telebot).
> **Главная архитектура:**
> Бот должен связывать личные сообщения от пользователей с Супергруппой администраторов, в которой включены Темы (Forum Topics). Для каждого нового пользователя бот должен создавать отдельную тему в группе админов. Чтобы бот не забывал связки пользователей и тем при перезагрузке, используй встроенную базу данных sqlite3.
> **База данных (SQLite):**
>  * Создай файл support.db.
>  * Таблица topics с двумя колонками: user_id (INTEGER, PRIMARY KEY) и topic_id (INTEGER).
>  * Напиши простые функции для инициализации БД, сохранения связи, поиска topic_id по user_id и поиска user_id по topic_id.
> **Логика работы со стороны ПОЛЬЗОВАТЕЛЯ:**
>  1. Бот ловит сообщения в личных чатах (chat.type == 'private'). Разрешенные content_types: текст, фото, видео, документы, голосовые.
>  2. Бот ищет user_id в БД.
>  3. Если пользователя нет в БД:
>    * Бот создает новую тему в админ-группе (через create_forum_topic). Название темы: "Имя_пользователя | ID".
>    * Сохраняет связь в БД.
>    * Отправляет в созданную тему системное сообщение: "Новое обращение от Имя (ID)".
>  4. Бот пересылает (forward_message) сообщение пользователя в эту конкретную тему (message_thread_id).
> **Логика работы со стороны АДМИНА:**
>  1. Бот ловит сообщения только из админ-группы (chat.id == ADMIN_GROUP_ID).
>  2. Бот проверяет, написано ли сообщение внутри темы (message.is_topic_message).
>  3. Бот берет message_thread_id и ищет по нему user_id в БД.
>  4. Если user_id найден:
>    * Если это текст, бот отправляет его пользователю от лица поддержки.
>    * Если это медиа (фото, стикер, файл, голосовое), бот копирует его пользователю через copy_message.
>  5. Оберни отправку ответа пользователю в блок try...except. Если пользователь заблокировал бота, отправь уведомление об ошибке обратно в тему админов.
> **Технические детали:**
>  * Вынеси TOKEN и ADMIN_GROUP_ID в начало скрипта в виде констант, чтобы я мог их легко заменить.
>  * Для запуска используй bot.polling(none_stop=True).
>  * Напиши код чисто, с подробными комментариями на русском языке.
